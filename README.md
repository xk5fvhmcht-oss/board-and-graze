# Omar's Board & Graze 🧀

A charcuterie board randomizer built for Dallas — pork-free by design, sourced from three local stores.

## Stores
- **Central Market** — 5750 E Lovers Ln, Dallas TX (anchor store, best cheese selection in Dallas)
- **Sara's Mediterranean Market** — 750 S Sherman St, Richardson TX (halal, Levantine, Persian, Middle Eastern)
- **Altin Grocery** — 1201 N Central Expy #19, Plano TX (Turkish specialist, 6,000+ products)

## Features
- 12 cultural themes with flag tiles (American, Latin, Mediterranean, French, Italian, Spanish, Greek, Turkish, Middle Eastern, North African, Persian, Levantine)
- Theme clash warnings for incompatible flavor combinations
- 3 board sizes: Small (1/cat), Medium (2/cat), Large (4/cat)
- 3 meal roles: Appetizer, Main Course, Grazing Table
- Headcount-aware serving sizes in both imperial and metric
- Per-category item count sliders (0 = exclude)
- Re-roll individual items
- "Don't show again" per item — persists across sessions
- Shopping list grouped by store with quantities
- Board supplies checklist
- PWA — installs to iOS home screen, works offline

## 212 items across 10 categories
Meats · Cheeses · Fruits · Crackers · Breads · Nuts · Spreads · Pickles · Vegetables · Extras

All items are **pork-free**.
